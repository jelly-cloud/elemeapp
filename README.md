#饿了么app

##主要依赖

- 基于vue.js
- 使用vue-cli搭建项目框架
- 使用vue-router进行页面路由切换
- 使用vue-resource进行http请求获取数据
- mock假数据存储在本地
- 使用stylus编写样式
- 使用eslint进行js代码的规范
- 使用better-scroll进行联动滑动

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
